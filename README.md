# 中古マンション価格トラッカー

不動産情報ライブラリの公開APIを利用して、特定地域の中古マンション価格を追跡するWebアプリケーションです。

## 機能

- 都道府県・市区町村別の中古マンション価格データの取得
- 価格推移グラフの表示（平均価格・平均㎡単価）
- 物件データの一覧表示とページング
- 高度なフィルタリング機能（期間、価格、面積、築年、間取り、構造）
- 欠損データの自動取得（2005年〜2024年の全期間）
- SQLiteデータベースへのデータ保存
- リアルタイム進捗表示

## セットアップ

1. 依存パッケージのインストール
```bash
npm install
```

2. 環境変数の設定
`.env`ファイルを編集し、不動産情報ライブラリAPIのキーを設定してください：
```
API_KEY=YOUR_API_KEY_HERE
```

APIキーは[不動産情報ライブラリ](https://www.reinfolib.mlit.go.jp/)で申請することで取得できます。

3. サーバーの起動
```bash
npm start
```

開発モード（自動リロード）で起動する場合：
```bash
npm run dev
```

4. ブラウザでアクセス
```
http://localhost:3000
```

## 使い方

1. 都道府県と市区町村を選択
2. 「データ取得・表示」ボタンをクリック
   - データベースに未取得のデータがある場合、自動的にAPIから取得
   - 進捗バーでデータ取得状況を確認
3. グラフと物件データ一覧が表示されます
4. フィルター機能を使用してデータを絞り込み
   - 期間範囲（年月）
   - 価格範囲（万円）
   - 面積範囲（㎡）
   - 築年範囲
   - 間取り
   - 構造
5. ページング機能で全ての物件データを閲覧

## 注意事項

このサービスは、国土交通省不動産情報ライブラリのAPI機能を使用していますが、提供情報の最新性、正確性、完全性等が保証されたものではありません。
